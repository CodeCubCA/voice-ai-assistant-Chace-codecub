- **How to capture**:

  3. Have at least 2-3 message exchanges visible in the chat
  4. Make sure the sidebar is visible with settings
  5. Take a full-window screenshot

## Screenshot Tips

- Use high resolution (at least 1280px wide)
- Ensure good contrast and readability
- Capture on a clean browser window (no extra tabs visible)
- Use PNG format for better quality
- Make sure all text is readable
- Show the app in a realistic usage scenario

## Taking Screenshots

### On macOS:
- Full screen: `Cmd + Shift + 3`
- Selected area: `Cmd + Shift + 4`
- Specific window: `Cmd + Shift + 4`, then press `Space`, then click the window

### On Windows:
- Full screen: `PrtScn` or `Windows + PrtScn`
- Selected area: `Windows + Shift + S`
- Specific window: `Alt + PrtScn`

### On Linux:
- Full screen: `PrtScn`
- Selected area: `Shift + PrtScn`
- Specific window: `Alt + PrtScn`

## After Capturing

1. Save all screenshots to this directory with the exact filenames listed above
2. Verify all images are PNG format
3. Check that images display correctly in the README
4. Optimize images if file sizes are too large (use tools like ImageOptim, TinyPNG, etc.)
